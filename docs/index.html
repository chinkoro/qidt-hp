アプリ

Complexlab

分子／錯体の3D可視化。距離・角度・配位を直感的に把握し、注釈画像や測定CSVを出力します。

Analysiscopilot

IR/NMR/XAFS/DSCなどの実測データをドラッグ&ドロップで整形。図・表・一次解釈・レポート草案を一括生成します。

理念 / Philosophy

見えない分子・界面の現象を「見て触れられる」道具に変える。研究者の直感と再現性を同時に高めるために、次の原則で2つのアプリを作っています。

  • 可視化は思考の道具：図と数表をワンクリックで。根拠（前処理パラメータ）も必ず残す。
  • 再現可能性 × 速度：同じ操作を同じ結果で再現。テンプレ出力で“待たせない”共有。
  • ローカル優先：研究データは端末内で完結。オフラインでも動作。
  • 研究者主導UI：最小の操作数、ショートカットとバッチ。現場のフローに合わせる。
  • 学びの積層：毎週「図1・式1・洞察1」。小さな発見を継続的に積む。
  • 誠実・敬意・共創：データの扱いと表現に誠実であること。異なる専門性を尊重すること。

なぜ 2 つ？（Before/After を分ける）

  • Complexlab（Before）：実験の前に、モデルで相互作用の当たりをつける。距離・配位を見て仮説を立てる。
  • Analysiscopilot（After）：実測後に、データをきれいに整え、図表と一次解釈で共有する。

この往復で、実験設計 → 測定 → 解釈 → 次の仮説のループを速く、確かに回します。

背景と個人的な動機（クリックで開く）

研究データがバラバラに散らばり、報告書作成が負担になって本質的な思考に時間が回らない——そんな課題感から出発しました。子どもの頃に憧れた「世界の魔法（=化学反応のメカニズム）」を、仲間と笑顔で探求できる環境を自分で作る。そのための最小の道具がこの2つです。

English tagline: Translate invisible molecular/interface phenomena into tangible tools. Built as local-first, reproducible utilities that accelerate the loop of hypothesis → measurement → interpretation.
